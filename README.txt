Willkommen!

Hier eine kleine Anleitung um das Wordle-Projekt zu nutzen.
Momentan muss das Wordle noch in einem Python Interpreten ausgeführt werden.
Falls noch keiner vorhanden in dem Ordner "Empfohlener_Interpreten" sind Installationsdateien für Windows, Mac, oder Linux Distros die von Debian abstammen.
Alternativ kann man den Order mit der Wordle.py Datei im Terminal öffnen und dann "python3 Wordle.py" Eingeben.
